🏎️ Autocar Making Project

이서현의 오늘 교육 내용 일지 정리

1교시수업 오리엔테이션

Autocar Making Project에 대한 전체적인 일정과 수업 방향성 소개

2교시개발 환경 설정 (1)

윈도우에서 리눅스를 쓸 수 있게 해주는 WSL2 설치 및 Ubuntu(Linux 기초) 학습, 코드 편집기인 VS Code 설치 및 연동 완료

3교시버전 관리 및 협업 도구

소스코드 관리를 위한 Git 설치, GitHub 계정 생성 및 원격 저장소(Repository) 만들기 실습

4교시Python 기초 개발

파이썬 언어의 특징과 개발 환경 확인 및 기초 구조 파악

5교시출력 함수 이해

화면에 데이터를 띄워주는 가장 기본적이고 중요한 print 함수 해석 및 실습

6교시변수와 자료형

데이터를 담는 바구니인 변수(Variable)의 개념과 숫자, 문자 등 다양한 자료형(Data Type) 학습

7교시연산자와 표현식

컴퓨터가 계산을 수행하도록 하는 사칙연산, 비교 연산 등 여러 연산자와 표현식 다루기

8교시문자열 포매팅

문자열을 더 효율적이고 가독성 좋게 출력하는 f-string 기법과 format() 함수 집중 실습
